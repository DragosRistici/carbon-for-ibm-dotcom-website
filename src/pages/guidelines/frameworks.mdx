## Why Carbon for IBM.com only support React and Web components?

Carbon for IBM.com comes with a variety of packages. While the main package used would be either React or Web Components, separate packages can be utilized as their own standalone functionality.
See [the Package section](https://www.ibm.com/standards/web/carbon-for-ibm-dotcom/developing#packages) on the Developing page for detail.
